给定一个字符串s，找到s中最长的回文子串。你可以假设s的最大长度为 1000。

思路

题目不难，需要理清思路。

首先，回文子串长度具有对称性，因此，回文子串可以从中心进行展开，而一个长度为n的字符串，对称中心可以有2n-1个。

而后从每个中心通过maxExpend(int flag, int center, String s)函数进行返回以此为中心时的回文子串长度，若返回值大于maxLength，进行更新，若返回值小于maxLength，进行下一个判断。

最后计算最长回文子串的start和end确定回文子串具体内容。
