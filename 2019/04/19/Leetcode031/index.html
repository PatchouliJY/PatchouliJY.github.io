Leetcode031：下一个排列

发表于 2019-04-19 | 分类于 Leetcode题解
字数统计: 702字 | 阅读时长 ≈ 2分钟

题目

实现获取下一个排列的函数，算法需要将给定数字序列重新排列成字典序中下一个更大的排列。

如果不存在下一个更大的排列，则将数字重新排列成最小的排列（即升序排列）。

必须原地修改，只允许使用额外常数空间。

以下是一些例子，输入位于左侧列，其相应输出位于右侧列。

思路

我真的非常菜，一开始的思路是将所有全排列顺序存储后进行搜索，并输出下一个，看了题解发现自己完全没有脑子（迫真）。

题解思路非常厉害。

首先，从数组尾开始进行搜索，直到第一个打破逆序的元素为止（即第一个nums[seek-1] >= nums[seek]），因为逆序是某个区间全排列中最后一项。

解释：比如[1,2,3,4,5]的全排列中[1,2,5,4,3]的下一个是[1,3,2,4,5]，[5,4,3]即为{3,4,5}元素全排列的最后一项。

现在，什么样的重新排列将产生下一个更大的数字？我们想要创建比当前更大的排列。因此，我们需要将数字nums[seek-1]替换为位于其右侧区域的数字中比它略大的数字（即右侧中最小的大于它的数字）。

由于我们从又开始搜索的序列为逆序数列，所以这个数可以通过从数组尾开始进行搜索，直到第一个大于它的元素为止。

而后将这两个数进行交换，并对nums[seek+1]向后所有的数进行升序排列（另一个序列的第一项，因为前一项的结尾是某个序列的最后一项，再进一步，则下个序列的末尾一定是某个序列的第一项）。
